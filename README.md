# 萌萌租約 Maid Cafe Game

這是一個使用純 HTML、CSS 和 JavaScript 開發的女僕咖啡廳經營模擬遊戲。

## 遊戲簡介

在《萌萌租約》中，你將扮演女僕咖啡廳的經營者。你的目標是在 30 天內透過經營咖啡廳賺取足夠的資金，支付每週不斷上漲的租金，並最終達成勝利目標。

## 遊戲玩法與規則

### 核心經營
- **服務顧客**：顧客進店後會坐到位置上，女僕需要完成「點餐」、「送餐」與「清理桌子」三個步驟。
- **名聲 (Reputation)**：名聲越高，客人進店的頻率越高。
- **平均評價 (Avg Rating)**：滿意度會影響店舖的評價。若平均星等低於 2.5 星，可能會面臨罰款。
- **女僕管理**：你可以招募不同的女僕，每位女僕都有自己的體力值。體力耗盡時需要休息。

### 遊戲目標
- **勝利條件**：在 **第 30 天** 結束前賺取超過 **$300,000**。
- **失敗條件**：如果無法支付每 7 天一次的租金，遊戲將直接結束。

### 定期支出：租金
- 遊戲每 **7 天** 會扣除一次租金。
- 租金會隨著天數增加而上漲，請務必保留足夠的庫存現金。

### 自動化升級
隨著遊戲進行，你可以購買自動化設備來減輕女僕的負擔：
- **掃地機器人 (Roomba)**：自動清理地面垃圾。
- **傳送帶 (Conveyor Belt)**：自動進行料理準備與送餐（雖然會稍微降低評價，但能大幅提升效率）。

## 開發技術
- **前端**：純 HTML5, CSS3, JavaScript (ES6+)。
- **數據控制**：JSON 格式的配置檔案（位於 `data/` 資料夾中）。

## 安裝與執行
1. 將專案連同目錄結構下載。
2. 使用任何網頁瀏覽器直接開啟 `index.html` 即可開始遊戲。
3. 推薦使用 VS Code 的 Live Server 擴充功能以獲得最佳體驗。
